Parameters

Encrypted ciphertext

\( c = G^m H^r \pmod n \), where \( r \in \mathbb{Z}_n \)

The computation is performed in two steps:

  • encrypt data
  • randomize ciphertext

Compute \( c = G^m \pmod n \)

If \( plaintext < 0 \), we remap it to the second half of the message space

If the private key is not available, we remap the message by computing the inverse of the ciphertext modulo \( n \) (which is equivalent to a homomorphic multiplication with \( -1 \)). Note that this operation is slower than remapping the value in plain text.
