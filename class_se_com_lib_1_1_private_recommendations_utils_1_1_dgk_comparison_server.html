Paillier::Ciphertext SeComLib::PrivateRecommendationsUtils::DgkComparisonServer::Compare ( const BigInteger &  rModTwoPowL) const

Interactive secure comparison.

Algorithm: \( t_{i + 1} = (1 - (a_i - b_i)^2) t_i + a_i (1 - b_i) \), for \( 0 \leq i < l + 1 \)

Note
In the paper, in Appendix C., it is stated that \( i < l + 1 \), but in D., \( a \) and \( b \) are \( l \) bit numbers. This is due to the data packing requirements, so l will be l + 1 for the data packing version
Parameters
rModTwoPowL\( r \pmod 2^l \)
Returns
encrypted result of the comparison: \( [0] \) or \( [1] \)

Compute \( t_l = t_{PSP} \oplus t_{SP} \)

\( t_{PSP} = \tau \) and \( t_{SP} = c \)

\( [t_l] = [t_{PSP}] \)

\( [t_l] = [1] [t_PSP]^{-1} \pmod n \)

Randomize \( \llbracket t_l \rrbracket \) and interact with the client to convert it to a Paillier encryption
