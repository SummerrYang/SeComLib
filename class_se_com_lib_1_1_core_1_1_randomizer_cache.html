Detailed Description

template<typename T_Container>
class SeComLib::Core::RandomizerCache< T_Container >

Randomizer cache.

This should be associated with a background thread that fills the cache once only a few unused items remain and a call to Pop should actually remove an element from the internal vector, but, for now, we pre-generate a constant quantity of random numbers and reuse them once they are depleted.
