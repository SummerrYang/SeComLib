Generates a random prime, guaranteed to have numberOfBits length.

Generates random numbers in the interval \( [0, 2^{numberOfBits - 1}) \). Shifts the integer to the interval \( [2^{numberOfBits - 1}, 2^{numberOfBits}) \) by setting the MSB. Repeats the process until the obtained number is prime.

void SeComLib::Core::RandomProviderGmp::Initialize ( RandomProviderBase< RandomProviderGmp > &  input)
static

Initializes the underlying random state from input.

Obtains a truly random seed via platform dependent means.

DO NOT use a function such as time(NULL) to generate it, because it does not provide a secure random seed (among other issues).

Initializes the internal GMP random generator state via gmp_randinit_default.

Seeds the random generator state with the random seed via gmp_randseed_ui.
