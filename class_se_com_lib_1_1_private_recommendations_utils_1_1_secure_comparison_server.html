Paillier::Ciphertext SeComLib::PrivateRecommendationsUtils::SecureComparisonServer::Compare ( const Paillier::Ciphertext &  similarityValue)

Interactive secure comparison of a similarity value with a preset threshold.

Compares \( [sim_{(A, B)}] \) with \( \delta \). If \( [sim_{(A, B)}] \geq \delta \) the result is 1. Otherwise it is 0.

Note
In the paper, it is stated that 0 is returned in the case of equality. As discussed with Thijs Veugen, this is a mistake.
