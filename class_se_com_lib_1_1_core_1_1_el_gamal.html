Decrypt number.

If \( plaintext \in [0, 2^t) \Rightarrow plaintext \geq 0 \) otherwise if \( plaintext \in [q - 2^t, q) \Rightarrow plaintext = plaintext - messageSpaceUpperBound \)

Parameters

Exceptions

Compute \( c.y * c.x^{-s} \pmod p \)

Shortcut: if \( c.y * c.x^{-s} \pmod p = 1 \), then \( c = [0] \)

\( m \in \mathbb{Z}_q \ (2^t, \lfloor q / 2 \rfloor + 2^t) \) is uniquely determined by \( g_q^m \pmod p \). Since we cannot determine m directly, we precompute all \( g_q^m \pmod p \) values, we store them in an std::map and we try to find m that matches \( c.y * c.x^{-s} \pmod p \)

Generates the ElGamal cryptosystem keys.

\( g \) is computed using algorithms described in the Handbook of Applied Cryptography by Alfred J. Menezes, Paul C. van Oorschot and Scott A. Vanstone (http://cacr.uwaterloo.ca/hac/).

Specifically, we use Algorithm 4.80, Chapter 4: Finding a generator of a cyclic group

INPUT: a cyclic group \( G \) of order \( n \), and the prime factorization \( n = p_1^{e_1} p_2^{e_2} \cdots p_k^{e_k} \)

OUTPUT: a generator \( \alpha \) of \( G \)

  1. Choose a random element \( \alpha \) in \( G \)
  2. For \( i \) from \( 1 \) to \( k \) do the following:
    1. Compute \( b \gets a^{n/p_i} \) (N.B. \( \pmod n \))
    2. If \( b = 1 \) then go to step 1.
  3. Return \( \alpha \).
Returns
Always true, for now.
Exceptions
std::runtime_errorparameter errors

bool SeComLib::Core::ElGamal::IsEncryptedZero ( const Ciphertext &  ciphertext) const

Determines if ciphertext contains an encryption of 0 or not.

If and only if \( m = 0 \), then \( c.y * c.x^{-s} \pmod p = 1 \).

This is faster than the actual decryption, since the table lookup is not required.
